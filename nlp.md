# Nlp

•NLP is a process of manipulation of natural language for example English, Hindi, French in form of audio or text, using some software.
•It is an automated process to extract required information from data by applyingmachine learning algorithms.
•Examples of NLP: POS Tagging, NER, Speech Recognition, Sentiment Analysis, Language Translation.
