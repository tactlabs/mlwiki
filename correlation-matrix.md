# Correlation Matrix

  It is a table that represents the numerical values of a correlation. This is the best and the most efficient way of representing the linear relationship between variables in a dataset. The interval for this correlation varies from a range of [-1,1].

  Types of Correlation :
* Positive, Negative or Zero Correlation
* Linear or Curvilinear Correlation
* Scatter Diagram Method
* Pearson's Product Moment Co-efficient of Correlation
* Spearman's Rank Correlation Coefficient
